Rescue stations

Data : https://storage.googleapis.com/montco-stats/tz.csv Here is data from accidents that happen in city XYZ.

Problem:

Evil zombies have destroyed all central rescue stations in the city. The state has statistics where emergency problems occurred in last year (data above) and they are not sure whether the emergency stations were placed efficiently. Note that ambulances were sent only to EMS problems. Therefore, they asked us to find places where they should build 5 (of the same size) emergency stations. What will their position be?

According to the map, now we can see in which town we have more EMS problems!¶

If we choice first top 5 towns from last result (that have most problems) to build EMS Station it will not be correct answer! why? because for example we have 2 towns which they have big EMS request but distance between their is not so far! (closed to each other)

What is the solution? If we find the distance between towns then it's possible to send other towns to main station (5 EMS station) and also another rule that is: we shouldn't build 2 EMS station to near to each other!

So, finding the distance between towns and making cluster based on distance could be the best solution for this problem.

NOTE: All distances are in real geographic system¶

# get distance by geopy library 
def cal_distance(from_twn,to_twn):
    return round(vincenty(from_twn, to_twn).kilometers)
